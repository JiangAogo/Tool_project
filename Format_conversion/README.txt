# 图像格式转换工具

## 简介

本工具用于批量将 `input` 文件夹中的图像文件转换为 PNG 格式，并将结果保存在 `output` 文件夹中。支持的输入格式包括：`.jpg`, `.jpeg`, `.png`, `.bmp`, `.gif`, `.webp`。本工具使用 Python 和相关库（`Pillow` 和 `tqdm`）实现图像处理，并提供进度条显示处理进度。

## 文件结构

- `input/` : 源文件夹，存放待处理的图像文件。
- `output/` : 目标文件夹，存放处理后的图像文件。
- `Format_conversion.py` : Python 源文件，包含图像转换代码。
- `README.txt` : 使用说明文件。
- `start.bat` : 一键启动批处理文件，用于启动 Python 环境并运行图像转换脚本。

## 环境依赖

1. Python 3.x 及以上版本
2. Pillow 库（用于图像处理）
3. tqdm 库（用于显示处理进度条）

## 使用方法

### 1. 准备工作

- 请确保已安装 Python 3.x 及以上版本。
- 将需要转换的图像文件放入 `input` 文件夹中。

### 2. 启动工具

- 双击运行 `start.bat` 文件。

  该批处理文件将执行以下操作：
  - 检查 Python 是否已安装。
  - 安装所需库（`Pillow` 和 `tqdm`），如果它们尚未安装。
  - 运行 `Format_conversion.py` 脚本，自动处理 `input` 文件夹中的所有图像文件，并将结果保存在 `output` 文件夹中。

### 3. 处理结果

- 所有处理后的图像文件将保存在 `output` 文件夹中。
- 批处理运行结束后，命令行窗口会显示 "图像处理完成！" 的信息。

### 注意事项

- 请确保 `input` 和 `output` 文件夹与 `Format_conversion.py` 和 `start.bat` 文件位于同一目录下。
- 如果运行过程中遇到任何错误，请确保已正确安装 Python 和所需的库。
